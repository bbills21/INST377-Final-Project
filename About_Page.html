About This Project

This project is a simple web application that provides information about anime. It includes a slideshow of top 25 anime, a filter for genres, and a list of anime titles, their MAL ranking, Their genres, and a basic description.

Additonally, this project has a home page with an ability to use voice commands to pull forth a specific top 25 anime along with its description genres, and MAL ranking.

The project is built using HTML, CSS, and JavaScript. The slideshow is created using a simple slider library, and the anime data is fetched from an external API.

Feel free to explore the different pages and enjoy the content! You can use the navigation bar to switch between pages.
